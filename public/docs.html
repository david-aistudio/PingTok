Developer Reference

The Universal TikTok Engine.

PingTok API provides a high-velocity, programmatic interface to extract metadata and download links from TikTok videos and slideshows.
Built for developers who need stability and speed.

Endpoints

GET REST API
/api/universal/resolve

Resolves a TikTok URL (Short or Canonical) into direct download links and metadata.

Query Parameters

Field Type Required Description
url string Yes The TikTok video URL (e.g. vt.tiktok.com/...)

Example Request

cURL 
curl -X GET \
  'https://pingtok.vercel.app/api/universal/resolve?url=https://vt.tiktok.com/ZSj...'
Node.js 
const res = await fetch(
  '/api/universal/resolve?url=...'
);
const data = await res.json();
console.log(data);
