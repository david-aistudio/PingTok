The Universal TikTok Engine.

PingTok API provides a high-velocity, programmatic interface to extract metadata and download links from TikTok videos and slideshows. Built for developers who need stability and speed.

Endpoints

GET /api/universal/resolve

Resolves a TikTok URL (Short or Canonical) into direct download links and metadata.

Query Parameters

Field Type Description
url string The TikTok video URL (e.g., https://vt.tiktok.com/...)

Example Request

curl -X GET \
  'https://pingtok.vercel.app/api/universal/resolve?url=https://vt.tiktok.com/ZSj...'
const res = await fetch(
  '/api/universal/resolve?url=...' 
);
const data = await res.json();
